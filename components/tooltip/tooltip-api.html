Settings

Name Default Args Description
element The component element.
settings object The component settings.
settings.content Takes title attribute or feed content. Can be a string or jQuery markup.
settings.offset {top:10,left:10} object How much room to leave.
settings.placement 'top' string Supports 'top'|'bottom'|'right'|'offset'.
settings.trigger 'hover' string Supports click and immediate and hover (and maybe in future focus).
settings.title string Title for Infor Tips.
settings.beforeShow string Call back for ajax tooltip.
settings.popover string force it to be a popover (no content).
settings.closebutton string Show X close button next to title in popover.
settings.isError false boolean Add error classes.
settings.isErrorColor false boolean Add error color only not description.
settings.tooltipElement string ID selector for an alternate element to use to contain the tooltip classes.
settings.parentElement object jQuery-wrapped element that gets. passed to the 'place' behavior as the element to place the tooltip against.
settings.keepOpen false boolean Forces the tooltip to stay open in situations where it would normally close.
settings.extraClass string Extra css class.
settings.placementOpt object Placement options.
settings.maxWidth string Toolip max width.

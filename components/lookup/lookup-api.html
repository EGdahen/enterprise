Settings

Name Default Args Description
element the base element
settings object incoming settings
settings.click function Provide a special function to run when the dialog opens to customize the interaction entirely.
settings.field 'id' string Field name to return from the dataset or can be a function which returns a string on logic
settings.title string Dialog title to show, or befault shows field label + "Lookup"
settings.buttons array Pass dialog buttons or Cancel / Apply
settings.options object Options to pass to the datagrid
settings.beforeShow function Call back that executes async before the lookup is opened.
settings.modalContent string Custom modal markup can be sent in here
settings.editable true boolean Can the user type text in the field
settings.autoApply true boolean If set to false the dialog wont apply the value on clicking a value.
settings.validator function A function that fires to let you validate form items on open and select
settings.autoWidth false boolean If true the field will grow/change in size based on the content selected.

Settings

Name Type Description
click function Provide a special function to run when the dialog opens to customize the interaction entirely.
field string Field name to return from the dataset or can be a function which returns a string on logic
title string Dialog title to show, or befault shows field label + "Lookup"
buttons array Pass dialog buttons or Cancel / Apply
options object Options to pass to the datagrid
beforeShow function Call back that executes async before the lookup is opened.
modalContent string Custom modal markup can be sent in here
editable boolean Can the user type text in the field
autoApply string If set to false the dialog wont apply the value on clicking a value.
validator function A function that fires to let you validate form items on open and select
autoWidth boolean If true the field will grow/change in size based on the content selected.

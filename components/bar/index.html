API Details

Settings

Name Default Args Description
element string The plugin element for the constuctor
settings string The settings element.
settings.dataset [] array The data to use in the line/area/bubble.
settings.isStacked true boolean Default is a single or stacked chart.
settings.isNormalized false boolean If true its a 100% bar chart
settings.isGrouped false boolean If true its a grouped bar chart
settings.showLegend true boolean If false the legend will not be shown.
settings.animate true true|false - will do or not do the animation, 'initial' will do only first time the animation.
settings.redrawOnResize true boolean If true, the component will not resize when resizing the page.
settings.formatterString string Use d3 format some examples can be found on http://bit.ly/1IKVhHh
settings.format true string The d3 axis format
settings.tooltip null string A tooltip for the whole chart
settings.useLogScale false boolean If true log scale is enabled.
settings.ticks null object Settings for the chart ticks. Can set ticks: {format: d3Format, number: n}
settings.showLines true boolean Show the in the axis lines or not.
settings.labelFactor 1.27 number How far out than the outer circle should the labels be placed, this may be useful to adjust for some labels.
settings.wrapWidth 60 number The number of pixels after which a label needs to be given a new line. You may want to change this based on label data.
settings.emptyMessage { title:(Locale?Locale.translate('NoData'):'No Data Available'), info:'', icon:'icon-empty-no-data'} object An empty message will be displayed when there is no chart data. This accepts an object of the form emptyMessage: `{title: 'No Data Available', info: 'Make a selection on the list above to see results' icon: 'icon-empty-no-data', button: {text: 'xxx', click: } }` Set this to null for no message or will default to 'No Data Found with an icon.'

Dataset Settings

  • name String - The name to show on the y axis for the bar
  • value Number - The raw data value for the bar.
  • tooltipString - The custom tooltip to show.

Extra Chart Settings

  • formatterString String - the d3 formatter string to show
  • showLines Boolean - Hide the lines going north south across the x axis
  • useLogScale Boolean - Use a logarithmic scale on the x axis. This can be used for skewed data.

Code Example

This example shows how to invoke a simple bar chart with a dataset controlling the values.


 var dataset = [{
      data: [{
          name: 'Category A',
          value: 373
      }, {
          name: 'Category B',
          value: 372
      }, {
          name: 'Category C',
          value: 236.35
      }],
      name: ''
    }];

  $('#bar-example').chart({type: 'bar', dataset: dataset});

Accessibility

  • The contrast and actual colors can be a concern for visibility impaired and color blind people. However, you can customize the color by passing higher contrast colors.

color: '#1a1a1a'
name: 'Component C'

Code Tips

You can override the tooltip by passing in a specific tooltip text value.


{type: 'bar-grouped', dataset: dataset, tooltip: 'Tooltip by attribute'}

Or if you have more dynamic requirements you can do this with an ajax callback.


$('#bar-grouped-example').chart({type: 'bar-grouped', dataset: dataset,
  tooltip: function(response) {
    //Ajax Call or async op
    setTimeout(function () {
      response('<strong>Tooltips Provide <br> Interesting Information</strong>');
    }, 400);
  }
});

Keyboard Shortcuts

  • Tab: You can tab into the chart area and through the legend values as each has a focus state.
  • Enter/Space: Will select the bar group the corresponds with the focus’d legend item.

Upgrading from 3.X

  • The bar chart was added in 3.6. From 3.6 the api is mostly compatible.

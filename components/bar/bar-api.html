Settings

Name Type Description
dataset array The data to use in the line/area/bubble.
isStacked boolean Default is a single or stacked chart.
isNormalized boolean If true its a 100% bar chart
isGrouped boolean If true its a grouped bar chart
showLegend boolean If false the legend will not be shown.
animate true|false - will do or not do the animation. 'initial' will do only first time the animation.
redrawOnResize boolean If true, the component will not resize when resizing the page.
formatterString string Use d3 format some examples can be found on http://bit.ly/1IKVhHh
format string The d3 axis format
tooltip string A tooltip for the whole chart
useLogScale boolean If true log scale is enabled.
ticks object Settings for the chart ticks. Can set ticks: {format: d3Format, number: n}
showLines boolean Show the in the axis lines or not.
labelFactor number How far out than the outer circle should the labels be placed, this may be useful to adjust for some labels.
wrapWidth number The number of pixels after which a label needs to be given a new line. You may want to change this based on label data.
emptyMessage object An empty message will be displayed when there is no chart data. This accepts an object of the form emptyMessage: {title: 'No Data Available', info: 'Make a selection on the list above to see results', icon: 'icon-empty-no-data', button: {text: 'xxx', click: <function>} } Set this to null for no message or will default to 'No Data Found with an icon.'

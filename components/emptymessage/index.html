Settings

Name Type Description
title string The Main text to show.
info string Longer paragraph text to show
icon string The name of the icon to use. See ./icons/example-empty-widgets.html for options.
button boolean The botton text and click event to add.
color string Defaults to 'graphite' but can also be azure. Later may be expanded to all personalization colors.

Code Example

Use an empty message when no data is present.


$('.empty-message').emptymessage({
  title: 'No Records Available',
  icon: 'icon-empty-no-data'
});

Responsive Guidelines

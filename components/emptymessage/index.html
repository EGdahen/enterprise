Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.title null string The Main text to show.
settings.info null string Longer paragraph text to show
settings.icon null string The name of the icon to use. See {@link ./icons/example-empty-widgets.html} for options.
settings.button null boolean The botton text and click event to add.
settings.color 'graphite' string Defaults to 'graphite' but can also be azure. Later may be expanded to all personalization colors.

Code Example

Use an empty message when no data is present.


$('.empty-message').emptymessage({
  title: 'No Records Available',
  icon: 'icon-empty-no-data'
});

Responsive Guidelines

  • The empty message will be centered in the container.

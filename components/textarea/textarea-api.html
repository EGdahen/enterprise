Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.autoGrow false boolean Will automatically expand the textarea to fit the contents when typing.
settings.autoGrowAnimate true boolean Will animate the textarea grow.
settings.autoGrowAnimateSpeed 200 number The speed of the animation.
settings.characterCounter true boolean Displays a counter that counts down from the maximum length allowed.
settings.printable true boolean Determines whether or not the text area can be displayed on a printed page.
settings.charRemainingText 'Characters Left' Text that will be used in place of the "remaining" text defaulting to a localized 'Characters Left'.
settings.charMaxText 'Character count maximum of' Text that will be used in place of the "Max" text. Defaults to a localized Version of 'Character count maximum of'.

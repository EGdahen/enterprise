Settings

Name Type Description
title string Title text or content shown in the message
isError boolean If true, will show title styled as an error with an error icon
message string The message content or text
width number Pass a specific with or defaults to auto
buttons object Array of buttons to add to the message (see modal examples as well)
cssClass string Extra Class to add to the dialog for customization.
returnFocus string JQuery Element selector to focus on return

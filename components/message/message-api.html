Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.title 'Message Title' string Title text or content shown in the message
settings.isError false boolean If true, will show title styled as an error with an error icon
settings.message 'Message Summary' string The message content or text
settings.width 'auto' number Pass a specific with or defaults to auto
settings.buttons null object Array of buttons to add to the message (see modal examples as well)
settings.cssClass null string Extra Class to add to the dialog for customization.
settings.returnFocus null string JQuery Element selector to focus on return

Settings

Name Default Args Description
element string The component element.
settings string The component settings.
settings.linearProgression false boolean The Main Application Name to display in the header. (Defaults to false)
settings.folderIconOpen 'caret-up' string A specific folder open icon. (Defaults to 'caret-up')
settings.folderIconClosed 'caret-down' string A specific folder close icon. (Defaults to 'caret-down')
settings.stepList '#step-list' boolean Determines whether or not to display device information (Browser, Platform, Locale, Cookies Enabled).
settings.stepLi '.js-step' string jQuery selector for the step elements.
settings.stepLink '.js-step-link' boolean jQuery selector for the step link elements.
settings.stepFolder '.js-step-folder' string jQuery selector for the step folder elements.
settings.btnPrev '.js-step-link-prev' string jQuery selector for the previous step button.
settings.btnNext '.js-step-link-prev' string jQuery selector for the next step button.
settings.beforeSelectStep function A callback (function or promise) that gives args: stepLink (the step link element) and isStepping (whether we are prev/next'ing or not).

Settings

Name Type Description
linearProgression Boolean The Main Application Name to display in the header. (Defaults to false)
folderIconOpen String A specific folder open icon. (Defaults to 'caret-up')
folderIconClosed String A specific folder close icon. (Defaults to 'caret-down')
stepList Boolean Determines whether or not to display device information (Browser, Platform, Locale, Cookies Enabled).
stepLi String jQuery selector for the step elements.
stepLink Boolean jQuery selector for the step link elements.
stepFolder String jQuery selector for the step folder elements.
btnPrev String jQuery selector for the previous step button.
btnNext String jQuery selector for the next step button.
beforeSelectStep Function A callback (function or promise) that gives args: stepLink (the step link element) and isStepping (whether we are prev/next'ing or not).

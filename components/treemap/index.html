API Details

Settings

Name Default Args Description
element string The plugin element for the constuctor
settings string The settings element.
settings.dataset array The data to use in the radar
settings.redrawOnResize true boolean If false, the component will not resize when resizing the page.
settings.margin object The margins of the SVG, which you may want to adjust depending on text location.
settings.colors array An array of colors used in sequence from front to end of the array.
settings.showLabel boolean If false then the percentage wont be shown in the blocks.
settings.labelFormatter string The d3 formatter function for the value label.
settings.showTitle boolean If true then the first name will be used for the title area.
settings.emptyMessage object An empty message will be displayed when there is no chart data. This accepts an object of the form emptyMessage: `{title: 'No Data Available', info: 'Make a selection on the list above to see results', icon: 'icon-empty-no-data', button: {text: 'xxx', click: } }` Set this to null for no message or will default to 'No Data Found with an icon.'

Accessibility

The contrast and actual colors can be a concern for visibility impaired and color blind people. However, you can customize the color by passing different contrast colors.

Code Tips

You can control the size of the treemap by setting the size of the parent element the treemap lives in. This may include possibly using an inset margin in some cases.

Settings

Name Type Description
min   -   if defined, provides a minimum numeric limit
max   -   if defined, provides a maximum numeric limit
step   -   if defined, increases or decreases the spinbox value by a specific interval whenever the control buttons are used.
validateOnInput boolean   -   If set to false, will only automatically correct the spinbox value after the spinbox has lost focus.

Settings

Name Default Args Description
element the base element
settings object incoming settings
settings.autocorrectOnBlur false boolean If true the input will adjust to the nearest step on blur.
settings.min null if defined, provides a minimum numeric limit
settings.max null if defined, provides a maximum numeric limit
settings.step null if defined, increases or decreases the spinbox value by a specific interval whenever the control buttons are used.
settings.validateOnInput true boolean If set to false, will only automatically correct the spinbox value after the spinbox has lost focus.

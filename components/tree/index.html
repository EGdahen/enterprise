Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.selectable 'single' string 'single' or 'multiple'.
settings.hideCheckboxes false boolean Only applies when `selectable` is set to 'multiple'.
settings.menuId if defined, will be used to identify a Context Menu by ID attribute in which to add nodes.
settings.useStepUI false boolean if `true`, turns this tree instance into a "Stepped" tree.
settings.folderIconOpen 'open-folder' string the icon used when a tree folder node is open.
settings.folderIconClosed 'closed-folder' string the icon used when a tree folder node is closed.
settings.sortable false boolean if `true`, allows nodes to become sortable.
settings.onBeforeSelect If defined as a function, fires that function as a callback before the selection on a node occurs.
settings.onExpand If defined as a function, fires that function as a node is expanded.
settings.onCollapse If defined as a function, fires that function as a node is collapsed.

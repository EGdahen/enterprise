Settings

Name Type Description
title string text that is displayed in the Toast's title.
message string text/HTML that's displayed in the Toast's body.
position string text that propagates into CSS classes that position the Toast in specific places. top left, bottom left, bottom right (center??)
audibleOnly boolean if true, causes the toast to be invisble on the screen, but still read out lout by screen readers.
progressBar boolean causes the toast to have a visible progress bar that will be completely disappeared when the toast should be removed.
timeout number the amount of time the toast should be present on-screen.

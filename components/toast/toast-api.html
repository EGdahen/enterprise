settings.message '(Content)' string Text/HTML that's displayed in the Toast's body.
settings.position 'top right' string text that propagates into CSS classes that position the Toast in specific places Can be top left, bottom left, bottom rightx
settings.audibleOnly false boolean if true, causes the toast to be invisble on the screen, but still read out lout by screen readers.
settings.progressBar true boolean causes the toast to have a visible progress bar that will be completely disappeared when the toast should be removed.
settings.timeout 6000 number the amount of time the toast should be present on-screen.

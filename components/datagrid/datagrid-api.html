settings.showDirty false boolean If true the dirty indicator will be shown on the rows

settings.allowOneExpandedRow true boolean Controls if you cna expand more than one expandable row.
settings.enableTooltips false boolean Process tooltip logic at a cost of performance
settings.disableRowDeactivation false boolean if a row is activated the user should not be able to deactivate it by clicking on the activated row
settings.sizeColumnsEqually false boolean If true make all the columns equal width
settings.expandableRow false boolean If true we append an expandable row area without the rowTemplate feature being needed.
settings.redrawOnResize false boolean If set to false we skip redraw logic on the resize of the page.
settings.exportConvertNegative false boolean If set to true export data with trailing negative signs moved in front.
settings.columnGroups null array An array of columns to use for grouped column headers.
settings.treeGrid false boolean If true a tree grid is expected so addition calculations will be used to calculate of the row children
settings.onPostRenderCell null Function A call back function that will fire and send you the cell container and related information for any cells cells with a component attribute in the column definition.
settings.onDestroyCell null Function A call back that goes along with onPostRenderCel and will fire when this cell is destroyed and you need noification of that.
settings.onEditCell null Function A callback that fires when a cell is edited, the editor object is passed in to the function
settings.onExpandRow null Function A callback function that fires when expanding rows. To be used. when expandableRow is true. The function gets eventData about the row and grid and a response function callback. Call the response function with markup to append and delay opening the row.
settings.emptyMessage {title:'No Data Available',infor:'',icon:'icon-empty-no-data'} object An empty message will be displayed when there is no rows in the grid. This accepts an object of the form emptyMessage: {title: 'No Data Available', info: 'Make a selection on the list above to see results', icon: 'icon-empty-no-data', button: {text: 'xxx', click: }} set this to null for no message or will default to 'No Data Found with an icon.'

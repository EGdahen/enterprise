Name Type Description

enableTooltips boolean Process tooltip logic at a cost of performance
disableRowDeactivation boolean if a row is activated the user should not be able to deactivate it by clicking on the activated row
sizeColumnsEqually boolean If true make all the columns equal width
expandableRow boolean If true we append an expandable row area without the rowTemplate feature being needed.
exportConvertNegative boolean If set to true export data with trailing negative signs moved in front.
columnGroups array An array of columns to use for grouped column headers.
treeGrid boolean : If true a tree grid is expected so addition calculations will be used to calculate of the row children
onPostRenderCell Function A call back function that will fire and send you the cell container and related information for any cells cells with a component attribute in the column definition.
onDestroyCell Function A call back that goes along with onPostRenderCel and will fire when this cell is destroyed and you need noification of that.
onEditCell Function A callback that fires when a cell is edited, the editor object is passed in to the function
onExpandRow Function A callback function that fires when expanding rows. To be used when expandableRow is true. The function gets eventData about the row and grid and a response function callback. Call the response function with markup to append and delay opening the row.
emptyMessage object An empty message will be displayed when there is no rows in the grid. This accepts an object of the form emptyMessage: {title: 'No Data Available', info: 'Make a selection on the list above to see results', icon: 'icon-empty-no-data', button: {text: 'xxx', click: }} set this to null for no message or will default to 'No Data Found with an icon.'

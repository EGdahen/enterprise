Settings

Name Default Args Description
element string The plugin element for the constuctor
settings string The settings element.
settings.dataset array The data to use in the radar
settings.redrawOnResize true boolean If false, the component will not resize when resizing the page.
settings.margin object The margins of the SVG, you may want to adjust depending on text location.
settings.levels 4 number How many levels or inner circles should there be drawn.
settings.maxValue 0 number What is the value that the biggest circle will represent
settings.labelFactor 1.27 number How far out than the outer circle should the labels be placed, this may be useful to adjust for some charts.
settings.wrapWidth 60 number The number of pixels after which a label needs to be given a new line. You may want to change this based on label data.
settings.showCrosslines true boolean Set to false to hide the cross line axes.
settings.showAxisLabels true boolean Set to false to hide percent labels.
settings.opacityArea 0.2 number The opacity value of the blobs. This is set to the correct Infor Style.
settings.dotRadius 3 number The size of the colored circles of each blog. Set to zero to remove dots.
settings.opacityCircles 0 number The opacity of the circles of each blob 0 or .1 are good values. This is set to the correct Infor Style.
settings.strokeWidth 1 number The width of the stroke around each blob. This is set to the correct Infor Style.
settings.roundStrokes true boolean If true the area and stroke will follow a round path (cardinal-closed).
settings.showCrosslines true boolean If false the axis lines will not be shown in the diagonals.
settings.showAxisLabels true boolean If false the axis labels will not be shown.
settings.axisFormatter '.0%' string D3 formatter to use on the axis labels
settings.colors array An array of colors to use.
settings.showTooltips true boolean If false no tooltips will be shown.
settings.tooltip object A setting that controls the tooltip values and format.
settings.tooltip.show 'value' string Controls what is visible in the tooltip, this can be value, label or percent or custom function.
settings.tooltip.formatter '.0%' object The d3.formatter string.
settings.showLegend true boolean If false the legend will not be shown.
settings.legendPlacement 'right' string Where to locate the legend. This can be bottom or right at the moment.
settings.emptyMessage object An empty message will be displayed when there is no chart data. This accepts an object of the form emptyMessage: `{title: 'No Data Available', info: 'Make a selection on the list above to see results', icon: 'icon-empty-no-data', button: {text: 'xxx', click: } }` Set this to null for no message or will default to 'No Data Found with an icon.'

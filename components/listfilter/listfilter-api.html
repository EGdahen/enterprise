Settings

Name Default Args Description
settings object incoming settings
settings.caseSensitive false boolean Set to true if searches ARE case sensitive
settings.filterMode 'startsWith' string Type of search can current be either 'startsWith' or 'contains'
settings.highlightMatchedText false boolean Inserts markup that appears to highlight text
settings.highlightCallback function If defined, will execute this code for highlighting text instead of the built-in highlighting code
settings.searchableTextCallback function If defined, will take each filterable item passed and return user-defined, searchable text content

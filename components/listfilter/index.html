Settings

Name Default Args Description
settings object incoming settings
settings.caseSensitive false boolean Set to true if searches ARE case sensitive
settings.filterMode 'startsWith' string Type of search can current be either 'startsWith' or 'contains'
settings.highlightMatchedText false boolean Inserts markup that appears to highlight text
settings.highlightCallback function If defined, will execute this code for highlighting text instead of the built-in highlighting code
settings.searchableTextCallback function If defined, will take each filterable item passed and return user-defined, searchable text content

Methods

Name Parameters Description Returns
filter list (array) = The array to search. term (string) = The term to look for.

Run the filter on the list for the given search term.

 false if filtering failed, or an array/jQuery of items that matched the filter.
updated settings (object) = incoming settings

Updates the ListFilter with new settings

 component instance object
destroy component instance object

Code Example

This is a component behavior used on listview, dropdown and multiselect other components. However, its API may be useful in applications for filtering elements.


// Make a List filter object
this.listfilter = new ListFilter({
  filterMode: 'contains'
});

// Return the reselts based on a term
var term = 'search for me';
results = this.listfilter.filter(list, term);

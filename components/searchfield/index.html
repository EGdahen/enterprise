Important Notes

Described below is the general API that is present in all searchfields. There is a specific component wrapper for searchfield components that reside inside of Toolbars entitled Toolbar Searchfield. These components are very different from regular searchfields and have an expanded API.

Settings

Name Type Description
resultsCallback function Callback function for getting typahead results on search.
allResultsCallback function Callback function for getting "all results".
showAllResults boolean If true the show all results link is showin in the list.
showGoButton boolean If true a go button is associated.
goButtonCopy string The text to use on the go button.
goButtonAction function If defined as a function, will fire this callback on the Go Button "click"
categories array If defined as an array, displays a dropdown containing categories that can be used to filter results.
categoryMultiselect boolean If true, creates a multiselectable categories list.
showCategoryText boolean If true, will show any available categories that are selected to the left of the Dropdown field.
source function Callback function for getting type ahead results.
template string The html template to use for the search list
clearable boolean Add an X to clear.

Code Example


<div class="field">
  <label for="searchfield">Search</label>
  <input id="searchfield" name="searchfield" class="searchfield" data-options= "{'clearable': 'true'}" placeholder="Type a search term"/>
</div>

Implementation Tips

  • Defining the source setting on Searchfield will cause it to invoke an Autocomplete. The source is not directly handled by the Searchfield component, and is passed directly into this new Autocomplete instance.

Accessibility

Keyboard Shortcuts

  • Tab Will tab into the search field in a forward direction.
  • Shift Tab Will tab out of the search field in a backward direction.
  • Alt + Delete (Mac) Will clear the contents of the field.
  • Ctrl + Backspace (PC) Will clear the contents of the field.
  • Up / Down Arrow When on a toolbar up and down will move to the next/previous object on the toolbar. This is done because the normal keys of left and right will navigate through the toolbar
  • Left / Right Arrow When on a toolbar left and right will navigate through the next/previous object on the toolbar. When a list is open it will navigate up and down a list.
  • Enter Should submit the search / form. You will need to handle this in your code.

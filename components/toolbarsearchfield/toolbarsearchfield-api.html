Name Default Args Description
element the base element
settings object incoming settings
settings.clearable true boolean If "true", provides an "x" button on the right edge that clears the field
settings.collapsible true boolean If "true", allows the field to expand/collapse on larger breakpoints when focused/blurred respectively
settings.collapsibleOnMobile true boolean If true, overrides `collapsible` only on mobile settings.

Settings

Name Default Args Description
element The plugin element for the constuctor
settings object The settings element.
settings.themes {} string Themes available for ColorPicker
settings.colors [] string An array of objects of the form. {label: 'Azure', number: '01', value: 'CBEBF4'} that can be used to populate the color grid.
settings.showLabel false string Show the label if true vs the hex value if false.
settings.editable true string If false, the field is readonly and transparent. I.E. The value cannot be typed only editable by selecting.
settings.uppercase true string If false, lower case hex is allowed. If true upper case hex is allowed. If showLabel is true this setting is ignored.
settings.colorOnly false string If true the field will be shrunk to only show the color portion.

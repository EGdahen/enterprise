Settings

Name Default Args Description
element object The plugin starting element.
settings object The settings to use.
settings.blockUI true string makes the element that Busy Indicator is invoked on unusable while it's displayed.
settings.text null string Custom Text To Show or Will Show Localized Loading....
settings.displayDelay 1000 string Number in miliseconds to pass before the markup is displayed. If 0, displays immediately.
settings.timeToComplete 0 boolean fires the 'complete' trigger at a certain timing interval. If 0, goes indefinitely.
settings.transparentOverlay false string If true, allows the "blockUI" setting to display an overlay that prevents interaction, but appears transparent instead of gray.
settings.overlayOnly false string If true, the busy indicator will only be the overlay.

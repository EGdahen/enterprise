Name Type Description
blockUI String   -   makes the element that Busy Indicator is invoked on unusable while it's displayed.
text String   -   Custom Text To Show or Will Show Localized Loading....

transparentOverlay String   -   If true, allows the "blockUI" setting to display an overlay that prevents interaction, but appears transparent instead of gray.

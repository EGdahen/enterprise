Settings

Name Default Args Description
element the base element
settings object incoming settings
settings.dataset array Array of data to feed the template
settings.template string Html Template String
settings.description string Audible Label (or use parent title)
settings.paging false boolean If true, activates paging
settings.pagesize 10 number If paging is activated, sets the number of listview items available per page
settings.pagingType 'list' string The paging type to use, this can be 'list', 'table' or 'firstlast'
settings.searchable false boolean If true, associates itself with a Searchfield/Autocomplete and allows itself to be filtered
settings.selectable 'single' selection mode, can be false, 'single', 'multiple' or 'mixed'
settings.selectOnFocus true boolean If true the first item in the list will be selected as it is focused.
settings.showCheckboxes true boolean If false will not show checkboxes used with multiple selection mode only
settings.hoverable true boolean If true the list element will show a hover action to indicate its actionable.
settings.emptyMessage string Text to go in emptyMessage.
settings.source If source is a string then it serves as the url for an ajax call that returns the dataset. If its a function it is a call back for getting the data asyncronously.
settings.forceToRenderOnEmptyDs false boolean If true list will render as an empty list with ul tag, but not any li tags in it.
settings.disableItemDeactivation false boolean If true when an item is activated the user should not be able to deactivate it by clicking on the activated item. They can only select another row.
settings.showPageSizeSelector false boolean If true the page size select will be shown when paging.

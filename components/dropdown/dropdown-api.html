Settings

Name Type Description
closeOnSelect boolean When an option is selected, the list will close if set to "true". List stays open if "false".
cssClass string Append an optional css class to dropdown-list
filterMode string Search mode to use between 'startsWith' and 'contains', false will not allow client side filter
noSearch boolean If true, disables the ability of the user to enter text in the Search Input field in the open combo box
showEmptyGroupHeaders boolean If true, displays headers in the list even if no selectable options are present underneath.
source boolean A function that can do an ajax call.
sourceArguments boolean If a source method is defined, this flexible object can be passed into the source method, and augmented with parameters specific to the implementation.
reloadSourceOnOpen boolean If set to true, will always perform an ajax call whenever the list is opened. If false, the first AJAX call's results are cached.
empty boolean Initialize Empty Value
delay boolean Typing Buffer Delay in ms
maxWidth number If set the width of the dropdown is limited to this pixel width. Fx 300 for the 300 px size fields. Default is size of the largest data.

Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.closeOnSelect true boolean When an option is selected, the list will close if set to "true". List stays open if "false".
settings.cssClass null string Append an optional css class to dropdown-list
settings.filterMode 'contains' string Search mode to use between 'startsWith' and 'contains', false will not allow client side filter
settings.noSearch false boolean If true, disables the ability of the user to enter text in the Search Input field in the open combo box
settings.showEmptyGroupHeaders false boolean If true, displays optgroup headers in the list even if no selectable options are present underneath.
settings.source boolean A function that can do an ajax call.
settings.sourceArguments {} boolean If a source method is defined, this flexible object can be passed into the source method, and augmented with parameters specific to the implementation.
settings.reloadSourceOnOpen false boolean If set to true, will always perform an ajax call whenever the list is opened. If false, the first AJAX call's results are cached.
settings.empty false boolean Initialize Empty Value
settings.delay 300 boolean Typing buffer delay in ms
settings.maxWidth null number If set the width of the dropdown is limited to this pixel width. Fx 300 for the 300 px size fields. Default is size of the largest data.
settings.placementOpts null object Gets passed to this control's Place behavior

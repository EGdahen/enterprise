API Details

Settings

Name Type Description
dataset array The data to use in the line/area/bubble.
isDonut boolean If true it renders as a donut chart.
animationSpeed number Controls the animation speed
animate true|false - will do or not do the animation. 'initial' will do only first time the animation.
redrawOnResize boolean If true, the component will not resize when resizing the page.
showTooltips boolean If false now tooltips will be shown even if there is tooltip values provided.
showLines boolean If false connector lines wont be shown
showLinesMobile boolean This defaults to false, when false and under 450px the lines will not be shown. If you still want lines at the lower breakpoint you can set this to true
showLegend boolean If false the legend will not be shown.
legendPlacement string Where to locate the legend. This can be bottom or right at the moment.
lines object A setting that controls the line values and format.
legend object A setting that controls the legend values and format.
tooltip object A setting that controls the tooltip values and format.

Code Example

This example shows how to invoke a pie bar chart with a dataset controlling the values.


  var pieData = [{
      data: [{
          name: 'Component A',
          value: 10.1,
          id: 'A',
          tooltip: 'Component A <b>{{percent}}</b>'
      }, {
          name: 'Component B',
          value: 12.2,
          id: 'B',
          tooltip: 'Component B <b>{{percent}}</b>'
      }, {
          name: 'Component C',
          value: 14.35,
          id: 'C',
          tooltip: 'Component C Is Very Cool<b>{{percent}}</b>'
      }]
    }];

  $('#pie-chart-example').chart({type: 'pie', dataset: pieData});

Accessibility

  • The contrast and actual colors can be a concern for visibility impaired and color blind people. However, you can customize the color by passing different contrast colors.

Code Tips

You can control the size of the donut chart by setting the size of the parent element the pie chart lives in. This may include possibly using an inset margin in some cases.

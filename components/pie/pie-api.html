Settings

Name Type Description
dataset array The data to use in the line/area/bubble.
isDonut boolean If true it renders as a donut chart.
animationSpeed number Controls the animation speed
animate true|false - will do or not do the animation. 'initial' will do only first time the animation.
showLegend boolean If false the legend will be hidden.
redrawOnResize boolean If true, the component will not resize when resizing the page.

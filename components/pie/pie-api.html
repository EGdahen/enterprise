Settings

Name Default Args Description
element The plugin element for the constuctor
settings object The component settings.
settings.dataset array The data to use in the line/area/bubble.
settings.isDonut false boolean If true it renders as a donut chart.
settings.animationSpeed 600 number Controls the animation speed
settings.animate true true|false - will do or not do the animation and 'initial' will do only first time the animation.
settings.redrawOnResize true boolean If true, the component will not resize when resizing the page. There is tooltip values provided. It will not be shown. If you still want lines at the lower breakpoint you can set this to true
settings.hideCenterLabel false boolean If false the center label will not be shown.
settings.showLines true boolean If false connector lines wont be shown
settings.showLinesMobile false boolean This defaults to false, when false and under 450px the lines
settings.lines object A setting that controls the line values and format.
settings.line.show 'value' string Controls the line value, this can be value, label or percent or custom function.
settings.line.formatter '.0f' string The d3.formatter string.
settings.showLegend true boolean If false the legend will not be shown.
settings.legendPlacement 'right' string Where to locate the legend. This can be bottom or right at the moment.
settings.legend object A setting that controls the legend values and format.
settings.legend.show 'label (percent)' string Controls what is visible in the legend, this can be value, value (percent), label or percent or your own custom function.
settings.legend.formatter '.0f' string The d3.formatter string.
settings.showTooltips true boolean If false now tooltips will be shown
settings.tooltip object A setting that controls the tooltip values and format.
settings.tooltip.show 'label (value)' string Controls what is visible in the tooltip, this can be value, label or percent or custom function.
settings.tooltip.formatter '.0f' string The d3.formatter string.

Settings

Name Default Args Description
element the base element for this component
settings object incoming settings
settings.addTabButton false boolean If set to true, creates a button at the end of the tab list that can be used to add an empty tab and panel
settings.addTabButtonCallback null function if defined as a function, will be used in-place of the default Tab Adding method
settings.appMenuTrigger false boolean If set to true, will force an App Menu trigger to be present on Non-Vertical Tabs implementatations.
settings.appMenuTriggerText string If defined, replaces the default "Menu" text used in the app menu trigger.
settings.ajaxOptions object if defined, will be used by any internal Tabs AJAX calls as the desired request settings.
settings.beforeActivate function If defined as a function, fires this before a tab is activated to allow a possible "veto" of the tab swap (SOHO-5250).
settings.containerElement null Defines a separate element to be used for containing the tab panels. Defaults to a `.tab-panel-container` element that is created if it doesn't already exist.
settings.changeTabOnHashChange false boolean If true, will change the selected tab on invocation based on the URL that exists after the hash.
settings.hashChangeCallback null function If defined as a function, provides an external method for adjusting the current page hash used by these tabs.
settings.lazyLoad true boolean if true, when using full URLs in tab HREFs, or when using Ajax calls, tabs will be loaded as needed instead of the markup all being established at once.
settings.moduleTabsTooltips false boolean if true, will display a tooltip on Module Tabs with cut-off text content.
settings.source null function If defined, will serve as a way of pulling in external content to fill tabs.
settings.sourceArguments {} object If a source method is defined, this flexible object can be passed into the source method, and augmented with parameters specific to the implementation.
settings.tabCounts false boolean If true, Displays a modifiable count above each tab.
settings.verticalResponsive false boolean If Vertical Tabs & true, will automatically switch to Horizontal Tabs on smaller breakpoints.

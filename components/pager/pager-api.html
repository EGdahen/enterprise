Settings

Name Default Args Description
element The component element.
settings object The component settings.
settings.componentAPI string If defined, becomes the definitive way to call methods on parent component.
settings.type 'list' string Different types of pagers list - just shows next and Previous and a listing of pages table shows next and previous and first and last with a page number input and page size selector used as the default for datagrid pageof - also shows next and previous and first and last with a page number input and page size selector used optionally for lists firstlast - shows next and previous and first and last with option to set showPageSizeSelector
settings.position 'bottom' string Can be on 'bottom' or 'top'.
settings.activePage 1 number Start on this page
settings.hideOnOnePage false boolean If true, hides the pager if there is only one page worth of results.
settings.source Function Call Back Function for Pager Data Source
settings.pagesize 15 number Can be calculated or a specific number
settings.pagesizes [15,25,50,75] array Array of numbers of the page size selector
settings.showPageSizeSelector false boolean If false will not show page size selector
settings.indeterminate false boolean If true will not show anything that lets you go to a specific

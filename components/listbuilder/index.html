Code Example

The list builder is structured as a listbuilder element that contains a standard toolbar structure followed by a listbuilder-content section that contains a standard listview.

When initialized with the $elem.listbuilder() plugin you pass in a data set that ineracts with the list view template. The toolbar buttons are automatically mapped to add, edit , delete , up , down functions via the option settings.


  <div class="listbuilder" >
    <div class="toolbar formatter-toolbar">
      ...
    </div>
    <div class="listbuilder-content">
      <div class="listview"></div>
    </div>
  </div>

Accessibility / Keyboard Shortcuts

  • ListView guidelines apply as this is conatains a listview.
  • Toolbar guidelines apply as this is conatains a toolbar.

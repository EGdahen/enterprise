Settings

Name Default Args Description
element The component element.
settings object The component settings.
settings.showTime false boolean If true the time selector will be shown.
settings.timeFormat string Format to use time section fx HH:mm, defaults current locale settings.
settings.minuteInterval number
settings.secondInterval number
settings.mode string Time picker options: 'standard', 'range', this controls the time picker.
settings.roundToInterval boolean In time picker mode, if a non-matching minutes value is entered, rounds the minutes value to the nearest interval when the field is blurred.
settings.dateFormat 'locale' string Defaults to current locale but can be
settings.placeholder false string Text to show in input element while empty.
settings.disable object Disable dates in various ways. For example `{minDate: 'M/d/yyyy', maxDate: 'M/d/yyyy'}`. Dates should be in format M/d/yyyy or be a Date() object or string that can be converted to a date with new Date().
settings.disable.dates array Disable specific dates. Example `{dates: ['12/31/2018', '01/01/2019'}`.
settings.disable.minDate Disable up to a minimum date. Example `{minDate: '12/31/2016'}`.
settings.disable.maxDate Disable up to a maximum date. Example `{minDate: '12/31/2019'}`.
settings.disable.dayOfWeek array Disable a specific of days of the week 0-6. Example `{dayOfWeek: [0,6]}`.
settings.disable.isEnable false boolean Inverse the disable settings. If true all the disable settings will be enabled and the rest will be disabled. So you can inverse the settings. For example if you have more non specific dates to disable then enable ect.
settings.showLegend false boolean If true a legend is show to associate dates.
settings.customValidation false boolean If true the internal validation is disabled.
settings.showMonthYearPicker false boolean If true the month and year will render as dropdowns.
settings.hideDays false boolean If true the days portion of the calendar will be hidden. Usefull for Month/Year only formats.
settings.advanceMonths 5 number The number of months in each direction to show in the dropdown for months (when initially opening)
settings.legend array Legend Build up for example `[{name: 'Public Holiday', color: '#76B051', dates: []}, {name: 'Weekends', color: '#EFA836', dayOfWeek: []}]`
settings.range object Range between two dates with various options.
settings.range.useRange false boolean Use range of two dates options.
settings.range.start Start date in range.
settings.range.end End date in range.
settings.range.separator ' - ' string Visual separator between two dates.
settings.range.minDays 0 number Minimum days to be in range.
settings.range.maxDays 0 number Maximum days to be in range.
settings.range.selectForward false boolean Range only in forward direction.
settings.range.selectBackward false boolean Range only in backward direction.
settings.range.includeDisabled false boolean Include disable dates in range of dates.
settings.calendarName string The name of the calendar to use in instance of multiple calendars. At this time only ar-SA and ar-EG locales have either 'gregorian' or 'islamic-umalqura' as valid values.
settings.useUTC false boolean If true the dates will use UTC format. This is only partially implemented https://jira.infor.com/browse/SOHO-3437

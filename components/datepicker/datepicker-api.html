Settings

Name Type Description
showTime Boolean   -   If true the time selector will be shown.
timeFormat String   -   The format to use on the time section fx HH:mm, defaults to current locale's settings.
mode String   -   Time picker mode: options: 'standard', 'range', this controls the avilable selections in the time picker.
roundToInterval Boolean   -   In time picker mode, if a non-matching minutes value is entered, rounds the minutes value to the nearest interval when the field is blurred.
dateFormat String   -   Defaults to current locale but can be overriden to a specific format
disable Boolean   -   Disabled Dates Build up. { 'dates' : [], 'minDate' : '', 'maxDate' : '', 'dayOfWeek' : [], 'isEnable' : false }
showMonthYearPicker Boolean   -   If true the month and year will render as dropdowns.
hideDays Boolean   -   If true the days portion of the calendar will be hidden. Usefull for Month/Year only formats.
customValidation Boolean   -   If true the internal validation is disabled.
advanceMonths Boolean   -   The number of months in each direction to show in the dropdown for months (when initially opening)
showLegend Boolean   -   If true a legend is show to associate dates.
legend Array   -   Legend Build up for example [{name: 'Public Holiday', color: '#76B051', dates: []}, {name: 'Weekends', color: '#EFA836', dayOfWeek: []}]
calendarName String   -   The name of the calendar to use in instance of multiple calendars. At this time only ar-SA and ar-EG locales have either 'gregorian' or 'islamic-umalqura' as valid values.

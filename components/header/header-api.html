Settings

Name Default Args Description
element the base element
settings object incoming settings
settings.demoOptions true boolean Used to enable/disable default SoHo Xi options for demo purposes
settings.useBackButton true boolean If true, displays a back button next to the title in the header toolbar
settings.useBreadcrumb false boolean If true, displays a breadcrumb on drilldown
settings.usePopupmenu false boolean If true, changes the Header Title into a popupmenu that can change the current page
settings.tabs null array If defined as an array of Tab objects, displays a series of tabs that represent application sections
settings.wizardTicks null array If defined as an array of Wizard Ticks, displays a Wizard Control that represents steps in a process
settings.useAlternate null boolean If true, use alternate background/text color for sub-navigation areas
settings.addScrollClass false boolean If true a class will be added as the page scrolls up and down to the header for manipulation. Eg: Docs Page.

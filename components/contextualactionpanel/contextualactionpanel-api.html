Settings

Name Type Description
id string The id to use for the CAP, or defaults to generated.
buttons array A list of buttons that will sit in the toolbar's Buttonset area.
title string String that sits in the toolbar's title field.
content content Pass content through to CAP.
initializeContent boolean Initialize content before opening with defaults.
trigger string Can be 'click' or 'immediate'.

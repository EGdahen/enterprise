Settings

Name Default Args Description
element string The component element.
settings string The component settings.
settings.id `contextual-action-modal-cnt` string The id to use for the CAP, or defaults to generated.
settings.buttons null array A list of buttons that will sit in the toolbar's Buttonset area.
settings.title 'Contextual Action Panel' string String that sits in the toolbar's title field.
settings.content null content Pass content through to CAP.
settings.initializeContent true boolean Initialize content before opening with defaults.
settings.trigger 'click' string Can be 'click' or 'immediate'.

Settings

Name Default Args Description
element The component element.
settings object The component settings.
settings.isStandalone true boolean On page(true)|on modal(false), used for some visual style only.
settings.standaloneClass 'standalone' string Css class if on page.
settings.allowedTypes '*' string Restrict file types(ie. 'jpg|png|gif') ['*' all types]
settings.maxFilesInProcess 99999 number Max number of files can be uploaded
settings.maxFileSize -1 number Max file size in bytes, -1 for unlimited
settings.fileName 'myfile' string Variable name to read from server
settings.isDisabled false boolean Make control disabled
settings.showBrowseButton true boolean Add way to browse files to upload
settings.send Function Method for send file to upload
settings.textDropArea string Text to show in drop area
settings.textDropAreaWithBrowse string Text to show in drop area when browse option true
settings.textBtnCancel string Hidden text for cancel button
settings.textBtnCloseError string Hidden text for error close button
settings.textBtnRemove string Hidden text for remove button
settings.errorAllowedTypes string Error text for allowed types
settings.errorMaxFileSize string Error text for max file size
settings.errorMaxFilesInProcess string Error text for max files in process

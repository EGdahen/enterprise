Settings

Name Type Description
appName string The Main Application Name to display in the header.
content string Additional text content to display at the top.
copyrightYear string The year displayed in the copyright, defaults to current year.
deviceSpecs boolean Determines whether or not to display device information. This information includes Browser, Platform, Locale and if Cookies are Enabled.
productName string Additional product name information to display.
useDefaultCopyright boolean Add the Legal Approved Infor Copy Right Text.
version string Semantic Version Number for example (4.0.0).

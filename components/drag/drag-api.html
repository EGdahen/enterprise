Settings

Name Type Description
axis string Constrains dragging to either axis. Possible values: null, 'x', 'y'
clone boolean Set to true to clone the object to drag. In many situations this is needed to break out of layout.
cloneCssClass string Css class added to clone element (defaults is 'is-clone')
clonePosIsFixed boolean If true cloned object will use css style "position: fixed"
cloneAppendTo string Selector to append to for the clone [ 'body'|'parent'|'jquery object' ] default:'body'
containment boolean Constrains dragging to within the bounds of the specified element or region. Possible values: "parent", "document", "window".
obstacle string jQuery Selector of object(s) that you cannot drag into,
underElements boolean If set to true will return list of elements that are underneath the drag element
containmentOffset string How close to the containment object should we be allowed to drag in position form. {left: 0, top: 0}

Methods

Name Parameters Description Returns
getElementsFromPoint x (number) = The x-coordinate of the Point. y (number) = The y-coordinate of the Point.

Get elements from given point.
updated settings (object) = the settings to update to.

Update the component and optionally apply new settings.
destroy

Detach all functionality and events.

Events

Name Args Description
dragend
event (object) = The jquery event object.
ui (object) = The dialog object
Fires after the drag is completed. Use this to remove / set drag feedback off.
drag
event (object) = The jquery event object.
ui (object) = The dialog object
Fires (many times) while dragging is occuring. Use this for DOM feedback but be careful about what you do in here for performance.
dragstart
event (object) = The jquery event object.
ui (object) = The dialog object
When the dragging is initiated. Use this to customize/style the drag/drop objects in the DOM.

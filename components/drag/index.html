Settings

Name Default Args Description
element The component element.
settings object The component settings.
settings.axis string Constrains dragging to either axis. Possible values: null, 'x', 'y'
settings.clone false boolean Set to true to clone the object to drag. In many situations this is needed to break out of layout.
settings.cloneCssClass 'is-clone' string Css class added to clone element (defaults is 'is-clone')
settings.clonePosIsFixed false boolean If true cloned object will use css style "position: fixed"
settings.cloneAppendTo string Selector to append to for the clone ['body'|'parent'|'jquery object'] default:'body'
settings.containment false boolean Constrains dragging to within the bounds of the specified element or region. Possible values: "parent", "document", "window".
settings.obstacle string jQuery Selector of object(s) that you cannot drag into,
settings.underElements false boolean If set to true will return list of elements that are underneath the drag element
settings.containmentOffset {left:0,top:0} object How close to the containment object should we be allowed to drag in position form. `{left: 0, top: 0}`

Events

Name Args Description
dragend
event (object) = The jquery event object.
ui (object) = The dialog object
Fires after the drag is completed. Use this to remove / set drag feedback off.
drag
event (object) = The jquery event object.
ui (object) = The dialog object
Fires (many times) while dragging is occuring. Use this for DOM feedback but be careful about what you do in here for performance.
dragstart
event (object) = The jquery event object.
ui (object) = The dialog object
When the dragging is initiated. Use this to customize/style the drag/drop objects in the DOM.

Code Example

The drag component is used internally in all components but its API my be useful for other situations. This basic example makes an element draggable and wont allow it to be dragged out of its direct parent.


$(draggableThing).drag({containment: 'parent'});

Accessibility

  • Dragging is NOT accessible. Try to provide alternate keyboard friendly ways to do the equivalent functionality.

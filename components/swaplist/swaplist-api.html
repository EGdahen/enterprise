Settings

Name Type Description
searchable boolean If true, associates itself with a Searchfield/Autocomplete and allows itself to be filtered
available array An array containing items for the available bucket.
selected array An array containing items for the selected bucket.
additional array An array containing items for the additional bucket.
availableClass string A class name linking the available root element.
selectedClass string A class name linking the selected root element.
additionalClass string A class name linking the additional root element.
availableBtn string A class name linking the available button element.
selectedBtnLeft string A class name linking the move left button element.
selectedBtnRight string A class name linking the move right button element.
additionalBtn string A class name linking the additional button element.
template string An Html String with the mustache template for the view.
draggable object An object containing boolean key/value to make container/s disable for dragging and moving items. Supported keys with draggable are "available", "selected", "additional".

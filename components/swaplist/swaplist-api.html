Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.searchable false boolean If true, associates itself with a Searchfield/Autocomplete and allows itself to be filtered
settings.available array An array containing items for the available bucket.
settings.selected array An array containing items for the selected bucket.
settings.additional array An array containing items for the additional bucket.
settings.availableClass '.available' string A class name linking the available root element.
settings.selectedClass '.selected' string A class name linking the selected root element.
settings.additionalClass '.full-access' string A class name linking the additional root element.
settings.availableBtn '.btn-moveto-selected' string A class name linking the available button element.
settings.selectedBtnLeft '.btn-moveto-left' string A class name linking the move left button element.
settings.selectedBtnRight '.btn-moveto-right' string A class name linking the move right button element.
settings.additionalBtn '.btn-moveto-selected' string A class name linking the additional button element.
settings.template string An Html String with the mustache template for the view.
settings.draggable object An object containing boolean key/value to make container/s disable for dragging and moving items. Supported keys with draggable are "available", "selected", "additional".

Settings

Name Default Args Description
element object The component element.
settings object The component settings.
settings.source [] string Defines the data to use, must be specified.
settings.sourceArguments {} string If a source method is defined, this flexible object can be passed into the source method, and augmented with parameters specific to the implementation.
boolean If defined, use this to draw the contents of each search result instead of the default draw routine.
settings.filterMode 'startsWith' string The matching algorithm, startsWith and contains are supported - false will not filter client side
settings.delay 300 boolean The delay between key strokes on the keypad before it thinks you stopped typing
settings.width null string Width of the open auto complete menu
settings.offset null string For the open menu, the left or top offset
settings.autoSelectFirstItem false string Whether or not to select he first item in the list to be selected
settings.resultsCallback function If defined, does not produce the results of the Autocomplete inside a popupmenu, instead piping them to a process defined inside this callback function.
settings.highlightMatchedText true boolean The highlightMatchText property.
settings.highlightCallback function The highlightCallback property.
settings.resultIteratorCallback function The resultIteratorCallback property.
settings.displayResultsCallback function The displayResultsCallback property.
settings.searchableTextCallback function The searchableTextCallback property.

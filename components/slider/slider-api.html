Settings

Name Default Args Description
element The DOM element
settings object incoming settings
settings.value [50] array An array with the slider values. Or one if a single value slider.
settings.min 0 number The minimum slider value.
settings.max 100 number The maximum slider value.
settings.range false boolean If true a range slider with two selectors is formed.
settings.step If added will be the number of slider steps to use.
settings.ticks [] array An array of the ticks to use for the steps
settings.tooltipContent Special customizable tooltip content.
settings.persistTooltip false boolean If true the tooltip will stay visible.

Methods

Name Parameters Description Returns
getIncrement

Gets a 10% increment/decrement as a value within the range of minimum and maximum values.

 nearest 10% increment number
increaseValue e (jQuery.Event) = jQuery click or keydown event. handle (Array) = represents a slider handle element. value (number) = target value - will be automatically determined if not passed. increment (number) = an integer that will be used as the amount to increment.

Increases the value of one of the slider handles, accounting for step value, percentage, etc. Also visually updates the handle on the visual part of the slider.
decreaseValue e (jQuery.Event) = jQuery click or keydown event. handle (Array) = element representing a slider handle. value (number) = target value - will be automatically determined if not passed. decrement (number) = an integer that will be used as the amount to decrement.

Decreases the value of one of the slider handles, accounting for step value, percentage, etc. Also visually updates the handle on the visual part of the slider.
updateTooltip handle () = element representing a slider handle.

If tooltips are active, updates the current placement and content of the Tooltip. If no handle argument is passed, this method simply hides both handles' tooltips.
value minVal (number) = the smaller handle's value maxVal (number) = the larger handle's value, if applicable

External Facing Function to set the value. Works as percent for now but need it on ticks. NOTE: Does not visually update the range. Use setValue() to do both in one swoop.

 both currently set handle values array

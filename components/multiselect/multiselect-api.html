Settings

Name Default Args Description
element the base element
settings object incoming settings
settings.filterMode 'contains' string The search mode to use, can be 'contains' or 'startsWith'
settings.maxSelected null number The max number of items which can be selected
settings.moveSelected 'all' string Move selected options in each group to just underneath their corresponding group headers.
settings.showEmptyGroupHeaders false boolean If true groups with no items will still show the empty group header.
settings.showSelectAll false boolean Show the select all button and text .
settings.source function The calback for ajax.

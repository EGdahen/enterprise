Settings

Name Type Description
filterMode string The search mode to use, can be 'contains' or 'startsWith'
maxSelected number The max number of items which can be selected
moveSelected string Move selected options in each group to just underneath their corresponding group headers.
showEmptyGroupHeaders boolean If true groups with no items will still show the empty group header.
showSelectAll boolean Show the select all text/option.
source Function The calback for ajax.

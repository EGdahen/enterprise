Tabs Test: Rename API Example

There's a built-in API in the Tabs Control that allows for renaming of Tab titles. The API exists because there are extra visual updates that need to happen to tabs when their contents change. Because the size of the tabs change, the Focus State and Animated Bar underneath the selected tab need to have their positions and sizes updated as well. The API serves this functionality without having to completely rebuild the tabs control.

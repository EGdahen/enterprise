Toolbar Test: Three Buttons Exactly

The Toolbar Control should only ever display three buttons at a time, including the "More Actions" Menu. However, when there are exactly THREE options present in the toolbar, a bug is possible where the third option automatically spills into the "More Actions" menu, creating a popupmenu that only has one item in it. This test checks to make sure we aren't creating single-option spillover menus.

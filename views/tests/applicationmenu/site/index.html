Notes About How This Test Works

There are three items in the Navigation Menu on this test:

  • Home: Working link with no accordion panes beneath. Clicking it anywhere should simply open the link.
  • Articles: Working link with accordion panes beneath. Clicking the "link" area should take activate the link. Clicking the expander (+/-) should expand/collapse the accordion pane.
  • Components: Inactive link with accordion panes beneath. Clicking anywhere on this header will expand/collapse the accordion pane.

The user will know that a link "worked" if an alert dialog is triggered when the link is clicked.

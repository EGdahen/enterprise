API Details

The control is initialized through the page initializer $('body').initialize('en-US'); or manually with $('.tabs').tabs();

The Tabs Control implements the following API methods that can be utilized by accessing the data object on its base element: $('.tabs').data('tabs').method();

Method Argument Description
add() string tabId, object options, integer atIndex Creates a new tab and tab panel with content, optionally at a specified index in the tab order. The tabId string must match an existing tab's href attribute, or no action will take place. In the options object, the following properties may be used:
  • string name: The Name of the Tab/Panel. Displayed inside the text of the Tab
  • boolean isDismissible: If true, creates a dismissible tab.
remove() string tabId Completely removes an existing tab and tab panel from the control. The tabId string must match an existing tab's href attribute, or no action will take place

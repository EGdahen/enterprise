About

js/about.js
new About(element: Object?, options: Object, appName: String)
Since: 4.2.0 The About Dialog Component displays information regarding the application. It should present the application name, version, copyright and any other information that would be useful for application users to present to support teams. Modal dialogs should not be used as a splash screen.
Parameters
element (Object? = this) The element to attach to (if manually calling the constructor)
options (Object)
Name Description
options.content String Additional Text content to display at the top.
options.copyrightYear String The year displayed in the copyright, defaults to current year.
options.deviceSpecs Boolean Determines whether or not to display device information (Browser, Platform, Locale, Cookies Enabled)
options.productName String Additional product name information to display
options.useDefaultCopyright Boolean Add the Legal Approved Infor Copy Right Txt
options.version String Semantic Version Number fx (4.0.0)
appName (String) The Main Application Name to display in the heading
Example
$('body').about({
  appName: 'SoHo Xi ',
  productName: 'Product Name',
  content: '<p>Fashionable components for fashionable applications.</p>'
});

splice

js/utils.js

The splice() method changes the content of a string by removing a range of characters and/or adding new characters.

splice(str: String, start: number, delCount: number, newSubStr: string): string
Parameters
str (String) The string that will be manipulated.
start (number) Index at which to start changing the string.
delCount (number) An integer indicating the number of old chars to remove.
newSubStr (string) The String that is spliced in.
Returns
string: A new string with the spliced substring.

DatePicker

js/datepicker.js

A component to support date entry. Users can either manually enter a date or select a date from a calendar. A secondary benefit is the ability to restrict the dates that are available for selection.

new DatePicker(element: Object?, options: Object?)
Parameters
element (Object? = this) The element to attach to (only when manually calling the constructor)
options (Object?)
Name Description
options.dateFormat string? (default 'locale') The format in which to display the date. Defaults to the current locales format but can be set to a custom format like 'yyyy-MM-dd' (iso8601 format).

Tooltip

js/tooltip.js

Tooltip and Popover Control

new Tooltip(element: Object, options: (Object | Function))
Parameters
element (Object)
options ((Object | Function))
Name Description
options.content (string | Function)? Takes title attribute or feed content. Can be a string or jQuery markup
options.offset Object? (default {top:10,left:10}) How much room to leave
options.placement string? (default top|bottom|right|offset)
options.trigger string? (default hover) supports click and immediate and hover (and maybe in future focus)
options.title string? Title for Infor Tips
options.beforeShow string? Call back for ajax tooltip
options.popover string? force it to be a popover (no content)
options.closebutton string? Show X close button next to title in popover
options.isError Boolean? (default false) Add error classes
options.isErrorColor Boolean? (default false) Add error color only not description
options.tooltipElement string? ID selector for an alternate element to use to contain the tooltip classes
options.parentElement Object? (default this.element) jQuery-wrapped element that gets passed to the 'place' behavior as the element to place the tooltip against.
options.keepOpen Boolean? (default false) Forces the tooltip to stay open in situations where it would normally close.
options.extraClass string? Extra css class
options.maxWidth string? Toolip max width
